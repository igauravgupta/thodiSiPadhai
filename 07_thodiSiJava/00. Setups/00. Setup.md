# Java — Setup

## 1. Install JDK

- Install an OpenJDK or Oracle JDK suitable for your OS.
  - Ubuntu / Debian: `sudo apt install openjdk-11-jdk` (or `openjdk-17-jdk`)
  - Fedora: `sudo dnf install java-11-openjdk-devel`
  - macOS (Homebrew): `brew install openjdk@11`
  - Windows: download installer from AdoptOpenJDK / Oracle and run it.
- After install, verify with:

```bash
java --version
javac --version
```

- Notes:
  - `java` is the Java runtime (JRE) that runs compiled classes.
  - `javac` is the Java compiler that compiles `.java` files to `.class` files.

## 2. Environment variables (optional but recommended)

- Set `JAVA_HOME` and add `bin` to `PATH` (example for bash/zsh):

```bash
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
export PATH="$JAVA_HOME/bin:$PATH"
```

- On Windows set `JAVA_HOME` in System Environment Variables and prepend `%JAVA_HOME%\bin` to `Path`.

## 3. JShell (REPL)

- Use JShell to quickly test code snippets without creating a class or boilerplate.
- Start JShell:

```bash
jshell
```

- Quick example session:

```
jshell> int sum(int a, int b) { return a + b; }
|  created method sum(int,int)
jshell> sum(2, 3)
$1 ==> 5
```

- Helpful JShell commands:
  - `/exit` — exit REPL
  - `/save filename.jsh` — save current snippets
  - `/open filename.jsh` — load saved snippets
  - Use arrow keys or Tab for history and completion.

## 4. Compile & run a simple program

1. Create `Hello.java`:

```java
public class Hello {
        public static void main(String[] args) {
                System.out.println("Hello, Java!");
        }
}
```

2. Compile and run:

```bash
javac Hello.java
java Hello
```

## 5. Troubleshooting

- If `java` or `javac` not found: ensure `JAVA_HOME` is set and `PATH` includes `$JAVA_HOME/bin`.
- Check installed versions to ensure compatibility between `javac` and runtime.

# Constructors and Destructors in C++

## Constructors

A **constructor** is a special member function of a class that is executed whenever a new object of that class is created. Its main purpose is to initialize the object's data members.

### Key Points

- Constructor name is same as the class name.
- No return type (not even `void`).
- Can be overloaded.
- Automatically called when an object is created.

### Types of Constructors

1. **Default Constructor**  
   No parameters.

   ```cpp
   class MyClass {
   public:
        MyClass() {
             // Initialization code
        }
   };
   ```

2. **Parameterized Constructor**  
   Takes arguments.

   ```cpp
   class MyClass {
   public:
        MyClass(int x) {
             // Use x for initialization
        }
   };
   ```

3. **Copy Constructor**  
   Initializes an object using another object of the same class.
   ```cpp
   class MyClass {
   public:
        MyClass(const MyClass &obj) {
             // Copy data from obj
        }
   };
   ```

### Constructor Example

```cpp
class Point {
     int x, y;
public:
     Point() : x(0), y(0) {} // Default
     Point(int a, int b) : x(a), y(b) {} // Parameterized
     Point(const Point &p) : x(p.x), y(p.y) {} // Copy
};
```

## Destructors

A **destructor** is a special member function that is executed when an object is destroyed. Its main purpose is to free resources allocated to the object.

### Key Points

- Destructor name is same as class name, prefixed with `~`.
- No parameters, no return type.
- Only one destructor per class (cannot be overloaded).
- Called automatically when object goes out of scope or is deleted.

### Destructor Example

```cpp
class MyClass {
public:
     ~MyClass() {
          // Cleanup code
     }
};
```

## Usage Example

```cpp
#include <iostream>
using namespace std;

class Demo {
public:
     Demo() { cout << "Constructor called\n"; }
     ~Demo() { cout << "Destructor called\n"; }
};

int main() {
     Demo obj; // Constructor called
     // Destructor called when obj goes out of scope
     return 0;
}
```

## Summary Table

| Feature     | Constructor              | Destructor              |
| ----------- | ------------------------ | ----------------------- |
| Name        | Same as class            | ~ClassName              |
| Return Type | None                     | None                    |
| Parameters  | Allowed (except default) | Not allowed             |
| Overloading | Yes                      | No                      |
| Purpose     | Initialize object        | Cleanup before deletion |

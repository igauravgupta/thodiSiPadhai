# Members of a Class in C++

In C++, a class is a blueprint for creating objects. The members of a class define its properties and behaviors. Class members are broadly categorized into **data members** and **member functions**.

## 1. Data Members

These are variables declared inside a class. They represent the attributes or state of the class.

### Types of Data Members

- **Private Data Members**: Accessible only within the class.
- **Protected Data Members**: Accessible within the class and its derived classes.
- **Public Data Members**: Accessible from outside the class.

```cpp
class Example {
private:
    int privateVar;
protected:
    int protectedVar;
public:
    int publicVar;
};
```

## 2. Member Functions

These are functions declared inside a class. They define the behavior of the class.

### Types of Member Functions

- **Private Member Functions**: Can be called only by other member functions within the class.
- **Protected Member Functions**: Accessible within the class and derived classes.
- **Public Member Functions**: Can be called from outside the class.

```cpp
class Example {
private:
    void privateFunc();
protected:
    void protectedFunc();
public:
    void publicFunc();
};
```

### Special Member Functions

- **Constructor**: Initializes objects of the class.
- **Destructor**: Cleans up when an object is destroyed.
- **Copy Constructor**: Initializes an object using another object of the same class.
- **Assignment Operator**: Assigns values from one object to another.

## 3. Static Members

- **Static Data Members**: Shared among all objects of the class.
- **Static Member Functions**: These functions are declared with the `static` keyword inside a class. Static member functions can only access static data members and other static member functions of the class, as they do not operate on a specific instance (object) and do not have access to the `this` pointer. They are called using the class name rather than an object.

```cpp
class Example {
public:
    static int staticVar;
    static void staticFunc();
};
```

## 4. Friend Functions

Not technically members, but can access private and protected members of the class.

```cpp
class Example {
    friend void friendFunc(Example&);
};
```

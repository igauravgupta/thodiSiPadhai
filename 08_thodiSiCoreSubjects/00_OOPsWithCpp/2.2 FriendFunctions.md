## What is a friend in C++?

friend is a keyword in C++.
It allows a non-member function or another class to access the private and protected members of a class.
By default, private and protected members of a class are not accessible outside the class.
But sometimes, we want external functions or classes to access them → for that we declare them as friends.

---

## Friend Function

### Definition

A friend function is a function not a member of the class, but it has the right to access private and protected data of that class.

### Syntax

```cpp
class ClassName {
	int data;
public:
	ClassName(int val) : data(val) {}
	friend void showData(ClassName obj); // declaration of friend function
};

// Friend function definition
void showData(ClassName obj) {
	cout << "Data = " << obj.data << endl; // can access private member
}
```

### Example

```cpp
class Box {
	int length;
public:
	Box(int l) : length(l) {}
	friend void printLength(Box b); // declare as friend
};

void printLength(Box b) {
	cout << "Length = " << b.length << endl; // direct access
}

int main() {
	Box b1(10);
	printLength(b1); // allowed
}
```

### Why use Friend Function?

To allow external functions to access private/protected data.
When operator overloading requires access to private members (like operator+, operator<<).
To provide tight coupling between two classes or between a class and a function.

## Friend Class

### Definition

A friend class is a class that can access private and protected members of another class.

### Syntax

```cpp
class B; // forward declaration

class A {
	int x;
public:
	A(int val) : x(val) {}
	friend class B; // declare class B as friend
};

class B {
public:
	void show(A a) {
		cout << "Accessing private A::x = " << a.x << endl;
	}
};
```

### Example

```cpp
class Account {
	int balance;
public:
	Account(int b) : balance(b) {}
	friend class Bank; // Bank is friend of Account
};

class Bank {
public:
	void showBalance(Account a) {
		cout << "Account balance: " << a.balance << endl;
	}
};

int main() {
	Account acc(5000);
	Bank SBI;
	SBI.showBalance(acc); // can access private "balance"
}
```

---

## Key Points about friend

Friendship is not mutual:
If class A is a friend of class B, that does not mean class B is a friend of class A.

Friendship is not inherited:
If class B is friend of class A, then derived classes of B are not automatically friends.

Friendship can be given to:

- A normal function
- A member function of another class
- An entire class

---

## Real-World Example

Imagine a Bank Account class where balance is private.
You may want an Auditor class or global function to access private balances for auditing or debugging → you make them friend.

---

## Summary

friend is a keyword.
Friend function: Not a class member, but can access private/protected data.
Friend class: Whole class gets access to another class’s private/protected members.
Useful in operator overloading, external function access, and tightly coupled classes.

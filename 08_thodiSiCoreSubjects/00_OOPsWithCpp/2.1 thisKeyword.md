## What is `this`?

this is an implicit pointer available inside all non-static member functions of a class.
It points to the object that invoked the member function.
In short, this → “the current object.”

---

## When is `this` Available?

Inside non-static member functions.
Not available in static functions (because they belong to the class, not any particular object).

---

## Uses of `this` in C++

### Accessing Data Members

```cpp
class Student {
    int age;
public:
    void setAge(int age) {
        this->age = age;   // distinguish between parameter and member
    }
};
```

Here, without this->, age = age; would assign the parameter to itself.

### Returning the Current Object (Chaining)

```cpp
class Number {
    int n;
public:
    Number(int n = 0) : n(n) {}
    Number& set(int val) {
        this->n = val;
        return *this;   // return reference to current object
    }
    void display() { cout << "n = " << n << endl; }
};

int main() {
    Number obj;
    obj.set(10).set(20).set(30); // method chaining
    obj.display();
}
```

### To Avoid Name Conflicts

If local variables shadow member variables, this resolves ambiguity.

```cpp
class Person {
    string name;
public:
    Person(string name) {
        this->name = name;   // differentiates between member and local variable
    }
};
```

### Passing Current Object as Argument

```cpp
class Box {
    int length;
public:
    Box(int l) : length(l) {}
    void compare(Box& other) {
        if (this->length > other.length)
            cout << "Current object is bigger\n";
        else
            cout << "Other object is bigger\n";
    }
};

int main() {
    Box b1(10), b2(20);
    b1.compare(b2);  // passes b1 implicitly via "this"
}
```

### Self-Referential Structures (Advanced OOP Design)

Sometimes used in operator overloading or when returning objects.

Example → Operator overloading:

```cpp
class Number {
    int value;
public:
    Number(int v) : value(v) {}
    Number& operator++() {  // prefix increment
        ++this->value;
        return *this;
    }
    void show() { cout << value << endl; }
};
```

## Special Notes

this is a const pointer → you cannot make this point somewhere else.
Type inside class → ClassName* const this.
Inside a const member function, this becomes a pointer to const:
const ClassName* const this
Not available in static functions.
Needed for fluent interfaces / chaining, operator overloading, and disambiguation.

---

## Summary

this → pointer to current object.
Used to resolve ambiguity, return current object, pass object to another function, and in operator overloading.
It’s automatically passed to all non-static member functions.

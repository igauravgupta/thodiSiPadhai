## Introduction to Inheritance

Inheritance is a fundamental concept in Object-Oriented Programming (OOP) that allows a class (derived class) to inherit properties and behaviors (data members and member functions) from another class (base class). It promotes code reusability and establishes a relationship between classes.

---

## Roles of Access Specifiers in Inheritance

Access specifiers (`public`, `protected`, `private`) determine how the members of the base class are accessible in the derived class:

- **Public Inheritance:**

  - Public members of base become public in derived.
  - Protected members of base become protected in derived.
  - Private members of base are not accessible directly.

- **Protected Inheritance:**

  - Public and protected members of base become protected in derived.
  - Private members of base are not accessible directly.

- **Private Inheritance:**
  - Public and protected members of base become private in derived.
  - Private members of base are not accessible directly.

**Example:**

```cpp
class Base {
public:
        int pub;
protected:
        int prot;
private:
        int priv;
};

class Derived : public Base {
        // pub is public, prot is protected, priv is inaccessible
};

class Derived2 : protected Base {
        // pub and prot are protected, priv is inaccessible
};

class Derived3 : private Base {
        // pub and prot are private, priv is inaccessible
};
```

## Types of Inheritance

| Type                     | Description                                                |
| ------------------------ | ---------------------------------------------------------- |
| Single Inheritance       | One derived class inherits from one base class.            |
| Multiple Inheritance     | One derived class inherits from more than one base class.  |
| Multilevel Inheritance   | A class is derived from another derived class.             |
| Hierarchical Inheritance | Multiple derived classes inherit from a single base class. |
| Hybrid Inheritance       | Combination of two or more types of inheritance.           |

## Inheritance in C++: Syntax & Example

```cpp
class Base {
public:
    int x;
    void show() { cout << "Base class"; }
};

class Derived : public Base {
public:
    int y;
    void display() { cout << "Derived class"; }
};
```

## Examples of All Types of Inheritance in C++

### 1. Single Inheritance

```cpp
class Animal {
public:
    void eat() { cout << "Eating"; }
};

class Dog : public Animal {
public:
    void bark() { cout << "Barking"; }
};
```

### 2. Multiple Inheritance

```cpp
class A {
public:
    void showA() { cout << "Class A"; }
};

class B {
public:
    void showB() { cout << "Class B"; }
};

class C : public A, public B {
public:
    void showC() { cout << "Class C"; }
};
```

### 3. Multilevel Inheritance

```cpp
class A {
public:
    void showA() { cout << "Class A"; }
};

class B : public A {
public:
    void showB() { cout << "Class B"; }
};

class C : public B {
public:
    void showC() { cout << "Class C"; }
};
```

### 4. Hierarchical Inheritance

```cpp
class Parent {
public:
    void showParent() { cout << "Parent class"; }
};

class Child1 : public Parent {
public:
    void showChild1() { cout << "Child1 class"; }
};

class Child2 : public Parent {
public:
    void showChild2() { cout << "Child2 class"; }
};
```

### 5. Hybrid Inheritance

```cpp
class A {
public:
    void showA() { cout << "Class A"; }
};

class B : public A {
public:
    void showB() { cout << "Class B"; }
};

class C : public A {
public:
    void showC() { cout << "Class C"; }
};

class D : public B, public C {
public:
    void showD() { cout << "Class D"; }
};
```

## Diamond Problem in Inheritance

The diamond problem occurs in multiple inheritance when two base classes inherit from the same grandparent class, and a derived class inherits from both base classes. This causes ambiguity as the derived class has two copies of the grandparent's members.

**Example:**

```cpp
class A {
public:
    void show() { cout << "Class A"; }
};

class B : public A {};
class C : public A {};
class D : public B, public C {};

int main() {
    D obj;
    // obj.show(); // Error: Ambiguous
}
```

**Solution:** Use `virtual` inheritance.

```cpp
class A {
public:
    void show() { cout << "Class A"; }
};

class B : virtual public A {};
class C : virtual public A {};
class D : public B, public C {};

int main() {
    D obj;
    obj.show(); // No ambiguity
}
```

## Common Interview Questions on Inheritance

| Question                                                                   | Short Answer                                                                              |
| -------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| What is inheritance?                                                       | Mechanism to acquire properties of another class.                                         |
| Types of inheritance in C++?                                               | Single, Multiple, Multilevel, Hierarchical, Hybrid.                                       |
| What is the diamond problem?                                               | Ambiguity due to multiple inheritance from a common base.                                 |
| How to solve diamond problem in C++?                                       | Use virtual inheritance.                                                                  |
| What is the difference between public, protected, and private inheritance? | Controls access level of base class members in derived class.                             |
| Can constructors be inherited?                                             | No, but base class constructor can be called from derived class.                          |
| What is object slicing?                                                    | When derived class object is assigned to base class object, extra members are sliced off. |
| What is the use of `virtual` keyword in inheritance?                       | To avoid multiple copies of base class in diamond problem.                                |

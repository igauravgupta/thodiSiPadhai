# useRef in React

## What is useRef?
`useRef` is a React Hook that returns a mutable ref object with a `.current` property. The ref object persists for the full lifetime of the component. Updating `.current` does not trigger a re-render.

## Syntax
```jsx
const ref = useRef(initialValue);
```

- `ref.current` holds the value (DOM node or any mutable value).
- Works with DOM refs (via `ref` prop) and as an instance-like container for mutable state.

## When to use (common use-cases)
- Access or manipulate a DOM element (focus, measure, scroll).
- Store mutable values that survive re-renders but don't need to trigger render (timers, IDs, counters).
- Keep previous props/state values across renders.
- Implement imperative methods or communicate with child components via `forwardRef`.
- Avoid stale closures by reading mutable values inside callbacks.

## Examples

### 1) Focus an input
```jsx
import React, { useRef } from 'react';

function FocusInput() {
    const inputRef = useRef(null);

    const focus = () => {
        inputRef.current?.focus();
    };

    return (
        <div>
            <input ref={inputRef} />
            <button onClick={focus}>Focus input</button>
        </div>
    );
}
```

### 2) Store a timer ID (no re-render)
```jsx
import React, { useRef, useEffect } from 'react';

function Timer() {
    const timerId = useRef(null);

    useEffect(() => {
        timerId.current = setInterval(() => {
            console.log('tick');
        }, 1000);
        return () => clearInterval(timerId.current);
    }, []);

    return <div>Timer running (check console)</div>;
}
```

### 3) Keep previous value across renders
```jsx
import React, { useRef, useEffect } from 'react';

function PrevValue({ value }) {
    const prevRef = useRef();
    useEffect(() => {
        prevRef.current = value;
    }, [value]);

    return <div>Current: {value}, Previous: {prevRef.current}</div>;
}
```

### 4) Forwarding refs to child components
```jsx
import React, { forwardRef, useRef } from 'react';

const Child = forwardRef((props, ref) => (
    <input ref={ref} placeholder="child input" />
));

function Parent() {
    const childRef = useRef(null);
    return (
        <>
            <Child ref={childRef} />
            <button onClick={() => childRef.current?.focus()}>Focus child</button>
        </>
    );
}
```

## Tips
- Use `useRef` for mutable state that should not cause re-renders.
- Prefer state (`useState`) when changes must reflect in the UI.
- When referencing DOM nodes, ensure the element exists (use optional chaining or null checks).

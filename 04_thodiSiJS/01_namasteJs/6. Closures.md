## What is a Closure?

A **closure** is a function that remembers its outer variables and can access them. In JavaScript, closures are created every time a function is created, at function creation time.

---

## How Closures Work

When a function is declared inside another function, the inner function forms a closure with its parent scope. This allows the inner function to access variables from the outer function even after the outer function has finished executing.

---

## Examples of Closures

```js
function outer() {
  let count = 0;
  function inner() {
    count++;
    return count;
  }
  return inner;
}

const counter = outer();
console.log(counter()); // 1
console.log(counter()); // 2
```

## Closures with `setTimeout`

Closures are commonly used with `setTimeout` to preserve variable values in asynchronous code. Without closures, all callbacks might reference the same variable value.

```js
for (var i = 1; i <= 3; i++) {
  setTimeout(function () {
    console.log(i); // Prints 4 three times
  }, 1000);
}
```

**Reason:**  
Here, the callback function inside `setTimeout` forms a closure over the variable `i`. Since `var` is function-scoped, all callbacks share the same `i`, which becomes 4 after the loop ends.

Using a closure, each callback captures its own value:

```js
for (var i = 1; i <= 3; i++) {
  (function (j) {
    setTimeout(function () {
      console.log(j); // Prints 1, 2, 3
    }, 1000);
  })(i);
}
```

**Reason:**  
An IIFE (Immediately Invoked Function Expression) is used to create a new scope for each iteration, so each callback gets its own copy of `j` (the current value of `i`).

With `let`, block scoping also solves this:

```js
for (let i = 1; i <= 3; i++) {
  setTimeout(function () {
    console.log(i); // Prints 1, 2, 3
  }, 1000);
}
```

**Reason:**  
`let` is block-scoped, so each iteration of the loop has its own `i`, and the closure captures the correct value for each callback.

## Constructor Function Example with Closures

You can use closures inside a constructor function to create private variables and methods. Here’s an example of a `Counter` constructor with `inc` and `dec` methods:

```js
function Counter() {
  let count = 0; // private variable

  this.inc = function () {
    count++;
    return count;
  };

  this.dec = function () {
    count--;
    return count;
  };
}

const myCounter = new Counter();
console.log(myCounter.inc()); // 1
console.log(myCounter.inc()); // 2
console.log(myCounter.dec()); // 1
```

**Explanation:**  
The `count` variable is not accessible from outside the `Counter` function. The `inc` and `dec` methods form closures over `count`, allowing controlled access and modification.

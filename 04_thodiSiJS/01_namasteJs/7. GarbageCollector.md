## What is Garbage Collection?

Garbage collection is the process of automatically freeing up memory that is no longer in use by the program.

## Why is Garbage Collection Needed?

JavaScript manages memory allocation for objects and variables. Without garbage collection, unused memory would not be released, leading to memory leaks and performance issues.

## How Garbage Collection Works in JS

JavaScript engines (like V8) periodically check which objects are still reachable and remove those that are not.

## Types of Garbage Collection Algorithms

| Algorithm          | Description                                    |
| ------------------ | ---------------------------------------------- |
| Mark-and-Sweep     | Marks reachable objects, sweeps unmarked ones. |
| Reference Counting | Frees objects with zero references.            |

## Best Practices

- Avoid creating unnecessary global variables.
- Nullify references when objects are no longer needed.
- Be cautious with closures and event listeners.

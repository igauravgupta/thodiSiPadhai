## 1. Array Declaration

### Using Array Literals (Recommended)

```js
const fruits = ["apple", "banana", "cherry"];
```

### Using the Array Constructor

```js
const numbers = new Array(1, 2, 3, 4);
```

- Prefer array literals for clarity and simplicity.

## 2. Common Array Methods for Manipulation

| Method       | Description                                   | Example                    |
| ------------ | --------------------------------------------- | -------------------------- |
| `push()`     | Add item(s) to the end                        | `arr.push('orange')`       |
| `pop()`      | Remove last item and return it                | `arr.pop()`                |
| `unshift()`  | Add item(s) to the beginning                  | `arr.unshift('grape')`     |
| `shift()`    | Remove first item and return it               | `arr.shift()`              |
| `splice()`   | Add/remove items at any position              | `arr.splice(1, 2, 'kiwi')` |
| `slice()`    | Copy part of array (does not modify original) | `arr.slice(1, 3)`          |
| `concat()`   | Merge arrays                                  | `arr.concat([4, 5])`       |
| `reverse()`  | Reverse the array in place                    | `arr.reverse()`            |
| `sort()`     | Sort the array                                | `arr.sort()`               |
| `includes()` | Check if array contains a value               | `arr.includes('apple')`    |
| `indexOf()`  | Get index of a value (first occurrence)       | `arr.indexOf('banana')`    |
| `join()`     | Join all elements into a string               | `arr.join(', ')`           |

## 3. Examples

```js
const arr = [1, 2, 3];
arr.push(4); // [1, 2, 3, 4]
arr.pop(); // [1, 2, 3]
arr.unshift(0); // [0, 1, 2, 3]
arr.shift(); // [1, 2, 3]
arr.splice(1, 1); // [1, 3] (removes 1 item at index 1)
const newArr = arr.slice(0, 2); // [1, 3]
```

## 4. Notes

- Arrays in JS are zero-indexed.
- Arrays can hold any data type (numbers, strings, objects, etc.).
- Most methods modify the original array except `slice`, `concat`, and a few others.
- Use `Array.isArray(obj)` to check if a variable is an array.
- A single array can contain different data types, e.g. `const mixed = [1, "two", true, { key: "value" }, [5, 6]];`

## Scenario: Modifying vs. Reinitializing a const Array

You can declare an array with `const` and change its elements, but you cannot reassign the variable to a new array.

```js
const colors = ["red", "green", "blue"];

// Modifying elements (allowed)
colors[0] = "yellow";
colors.push("purple");
console.log(colors); // ['yellow', 'green', 'blue', 'purple']

// Reinitializing (not allowed)
colors = ["black", "white"]; // ❌ TypeError: Assignment to constant variable.
```

**Explanation:**

- `const` prevents reassignment of the variable, but does not make the array immutable.
- You can change, add, or remove elements, but you cannot assign a new array to the same variable.

## The `forEach()` Method

The `forEach()` method executes a provided function once for each array element.

**Syntax:**

```js
array.forEach(function (element, index, array) {
  // code to execute for each element
});
```

- `element`: The current element being processed
- `index` (optional): The index of the current element
- `array` (optional): The array being traversed

**Example:**

```js
const fruits = ["apple", "banana", "cherry"];
fruits.forEach(function (fruit, idx) {
  console.log(idx + ": " + fruit);
});
// Output:
// 0: apple
// 1: banana
// 2: cherry
```

**Notes:**

- `forEach()` does not return a value (always returns undefined).
- It cannot be stopped or broken (unlike a regular for loop).
- Use it for side effects (e.g., logging, updating UI), not for building new arrays.

## The `map()` Method

The `map()` method creates a new array by applying a function to each element of the original array. It does not modify the original array.

**Syntax:**

```js
const newArray = array.map(function (element, index, array) {
  // return new value for each element
});
```

- `element`: The current element being processed
- `index` (optional): The index of the current element
- `array` (optional): The array being traversed

**Example:**

```js
const numbers = [1, 2, 3, 4];
const squares = numbers.map(function (num) {
  return num * num;
});
console.log(squares); // [1, 4, 9, 16]
```

**When to use `map()`?**

- Use `map()` when you want to transform each element of an array and create a new array with the results.
- Do NOT use `map()` for side effects (like logging or updating UI)—use `forEach()` for that.

**forEach vs. map:**

- `forEach()` is for executing a function for each element (side effects), returns undefined.
- `map()` is for transforming data and building a new array, returns the new array.

## The `find()` and `findIndex()` Methods

### `find()`

Returns the first element in the array that satisfies the provided testing function. If no element matches, returns `undefined`.

**Syntax:**

```js
const result = array.find(function (element, index, array) {
  // return true to find the element
});
```

**Example:**

```js
const users = [
  { id: 1, name: "Alice" },
  { id: 2, name: "Bob" },
  { id: 3, name: "Charlie" },
];
const user = users.find((u) => u.id === 2);
console.log(user); // { id: 2, name: 'Bob' }
```

### `findIndex()`

Returns the index of the first element that satisfies the provided testing function. If no element matches, returns `-1`.

## The `includes()` Method

Checks if an array contains a certain value. Returns `true` if found, otherwise `false`.

**Syntax:**

```js
array.includes(valueToFind, fromIndex);
```

- `valueToFind`: The value to search for
- `fromIndex` (optional): The position to start searching from

**Example:**

```js
const fruits = ["apple", "banana", "cherry"];
console.log(fruits.includes("banana")); // true
console.log(fruits.includes("grape")); // false
```

**When to use?**

- Use `includes()` to check if a value exists in an array (simple existence check).

---

## Quick Comparison: `includes()` vs `filter()`

| Method     | Returns            | Use Case                              |
| ---------- | ------------------ | ------------------------------------- |
| `includes` | `true`/`false`     | Check if a value exists in the array  |
| `filter`   | New array (subset) | Get all elements matching a condition |

**Example:**

```js
const arr = [1, 2, 3, 4, 5];
console.log(arr.includes(3)); // true
const evens = arr.filter((x) => x % 2 === 0);
console.log(evens); // [2, 4]
```

## Difference Between `slice()` and `splice()`

| Method     | Modifies Original? | Returns       | Use Case                               |
| ---------- | ------------------ | ------------- | -------------------------------------- |
| `slice()`  | No                 | New array     | Copy part of array (non-destructive)   |
| `splice()` | Yes                | Removed items | Add/remove/replace items (destructive) |

### `slice()`

- Does NOT change the original array.
- Syntax: `array.slice(start, end)` (end not included)
- Example:
  ```js
  const arr = [1, 2, 3, 4, 5];
  const part = arr.slice(1, 4); // [2, 3, 4]
  console.log(arr); // [1, 2, 3, 4, 5]
  ```

### `splice()`

- Changes the original array.
- Syntax: `array.splice(start, deleteCount, item1, item2, ...)`
- Example:
  ```js
  const arr = [1, 2, 3, 4, 5];
  const removed = arr.splice(1, 2, "a", "b"); // removes 2 items at index 1, inserts 'a', 'b'
  console.log(arr); // [1, 'a', 'b', 4, 5]
  console.log(removed); // [2, 3]
  ```

**Summary:**

- Use `slice()` to copy/extract without changing the original array.
- Use `splice()` to add/remove/replace items and change the original array.

## The `filter()` Method

Creates a new array with all elements that pass the test implemented by the provided function.

**Syntax:**

```js
const newArray = array.filter(function (element, index, array) {
  // return true to keep the element
});
```

- `element`: The current element being processed
- `index` (optional): The index of the current element
- `array` (optional): The array being traversed

**Example:**

```js
const numbers = [1, 2, 3, 4, 5, 6];
const even = numbers.filter((num) => num % 2 === 0);
console.log(even); // [2, 4, 6]
```

**When to use?**

- Use `filter()` when you want to select a subset of elements from an array based on a condition.

**Syntax:**

```js
const idx = array.findIndex(function (element, index, array) {
  // return true to find the index
});
```

**Example:**

```js
const numbers = [5, 12, 8, 130, 44];
const idx = numbers.findIndex((num) => num > 10);
console.log(idx); // 1 (12 is the first > 10)
```

**When to use?**

- Use `find()` when you need the actual element (object/value) that matches a condition.
- Use `findIndex()` when you need the index (position) of the matching element.

## The `filter()` Method

The `filter()` method creates a new array with all elements that pass the test implemented by the provided function. The original array is not modified.

**Syntax:**

```js
const filteredArray = array.filter(function (element, index, array) {
  // return true to keep the element
});
```

- `element`: The current element being processed
- `index` (optional): The index of the current element
- `array` (optional): The array being traversed

**Example:**

```js
const numbers = [10, 15, 20, 25, 30];
const greaterThan20 = numbers.filter((num) => num > 20);
console.log(greaterThan20); // [25, 30]
```

**When to use?**

- Use `filter()` when you want to select a subset of elements from an array based on a condition.

## The `reduce()` Method

The `reduce()` method executes a reducer function on each element of the array, resulting in a single output value. It is commonly used for summing values, accumulating results, or transforming arrays into objects.

**Syntax:**

```js
const result = array.reduce(function (accumulator, currentValue, index, array) {
  // return updated accumulator
}, initialValue);
```

- `accumulator`: The accumulated value returned by the last execution
- `currentValue`: The current element being processed
- `index` (optional): The index of the current element
- `array` (optional): The array being traversed
- `initialValue` (optional but recommended): The initial value for the accumulator

**Example:**

```js
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((acc, num) => acc + num, 0);
console.log(sum); // 10
```

**When to use?**

- Use `reduce()` when you need to combine all elements of an array into a single value (sum, product, object, etc.).
- It is powerful for advanced data transformations.
